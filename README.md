# Telegram Bot for Bitrix24 Deals

Простой Telegram-бот, который принимает данные от пользователя и создает сделки в Bitrix24.  
Сейчас проект запускается локально и подходит для разработки, тестирования и демонстрации базового сценария интеграции Telegram ↔ Bitrix24.

## Возможности

- запуск Telegram-бота локально;
- прием данных от пользователя через чат;
- формирование сделки на основе введенной информации;
- отправка сделки в Bitrix24 через API;
- простая логика без сложной инфраструктуры.

## Стек

- Python
- aiogram
- Bitrix24 REST API
- HTTP-клиент для запросов к Bitrix24

## Как это работает

1. Пользователь пишет боту в Telegram.
2. Бот последовательно запрашивает нужные данные.
3. После заполнения данных бот отправляет запрос в Bitrix24.
4. В Bitrix24 создается новая сделка.
